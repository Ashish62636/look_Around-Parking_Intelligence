# look_Around Dashboard Frontend 🗺️

This directory contains the WebGL-accelerated React + TypeScript + deck.gl dashboard for the Bengaluru Parking Intelligence system.

##  Quick Start

1. Ensure the backend FastAPI server is running on port 8000.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Access the dashboard at http://localhost:5173
